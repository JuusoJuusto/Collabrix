Q: What's the difference between Firebase emails and SMTP emails?
A:

   - Password reset emails
   - Sent by Firebase automatically
   - Uses Firebase servers

   - Welcome emails with custom design
   - 2FA verification codes
   - Server invitations
   - Custom notifications
   - Sent by your server using Gmail SMTP
