# Collabrix v2.1.0 - Major Feature Update

## 🎉 What's New

Collabrix v2.1.0 brings MASSIVE improvements that make it truly distinct from Discord with unique features and a beautiful new design!

## ✨ New Features

### 1. **AI Assistant** 🤖
- Built-in AI helper accessible with one click
- Summarize conversations instantly
- Get reply suggestions
- Translate messages on the fly
- Smart context-aware assistance

### 2. **Message Moods** 😊
- Express yourself with 5 different moods:
  - 😊 Happy
  - 😢 Sad
  - 🎉 Excited
  - 😠 Angry
  - 😐 Neutral
- Mood indicators appear next to messages
- Set mood before sending each message

### 3. **Important Messages** ⚡
- Mark messages as important with Ctrl+I
- Important messages get highlighted with yellow border
- Lightning bolt icon for quick identification
- Perfect for announcements and critical info

### 4. **Upvote/Downvote System** 👍👎
- Reddit-style voting on messages
- See vote counts in real-time
- Helps surface the best content
- Community-driven content curation

### 5. **Enhanced Reactions** ✨
- 10 quick reactions (up from 8)
- Smooth reaction picker popup
- Click any message to add reactions
- See who reacted to each message

### 6. **Poll Creator** 📊
- Create polls directly in chat
- Add unlimited poll options
- Visual poll interface
- Real-time voting results

### 7. **Advanced Search** 🔍
- Instant message search with Ctrl+K
- Filter messages as you type
- Highlight search results
- Fast and responsive

### 8. **Pinned Messages** 📌
- Pin important messages with one click
- Quick access to pinned messages (Ctrl+P)
- Dedicated pinned messages view
- Never lose important info

### 9. **Events Calendar** 📅
- Schedule and view events (Ctrl+E)
- Event reminders
- Integrated calendar view
- Perfect for team coordination

### 10. **Reply Threading** 💬
- Reply to specific messages
- See reply context in thread
- Visual reply indicators
- Better conversation flow

- **Ctrl+K**: Quick search
- **Ctrl+P**: View pinned messages
- **Ctrl+E**: Open events calendar
- **Ctrl+I**: Mark message as important
- **Escape**: Close all modals
