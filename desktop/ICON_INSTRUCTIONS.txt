ICON SETUP INSTRUCTIONS
=======================

You need to add an icon.png file to this folder.

Requirements:
- File name: icon.png
- Size: 512x512 pixels (recommended)
- Format: PNG with transparency
- Design: Collabrix logo or ðŸ’¬ emoji

Where to get an icon:
1. Create one in Photoshop/Figma
2. Use an online icon generator
3. Convert emoji to PNG: https://emoji.gg/
4. Use AI image generator

Quick Option:
- Download a Discord-style icon from flaticon.com
- Resize to 512x512
- Save as icon.png in this folder

After adding icon.png:
- Rebuild the app: npm run build:win
- The icon will appear in the installer and app
