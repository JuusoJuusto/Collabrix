QUICK ICON SETUP
================

You need icon.ico and icon.png files.

EASIEST WAY:
1. Go to: https://www.icoconverter.com/
2. Upload any image (logo, emoji, etc.)
3. Download as .ico
4. Save as "icon.ico" in this folder
5. Also save a PNG version as "icon.png"

OR use online emoji converter:
1. Go to: https://emoji.gg/
2. Search for "chat" or "message"
3. Download PNG
4. Convert to ICO at icoconverter.com
5. Save both files here

Then run: build-windows.bat
