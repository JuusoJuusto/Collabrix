ICON FILES REQUIRED BEFORE BUILDING
====================================

You need TWO icon files in this folder:

1. icon.ico (Windows icon, 256x256 pixels)
2. icon.png (PNG icon, 512x512 pixels)

HOW TO CREATE ICONS:
--------------------

Option 1 - Use an online converter:
  1. Create or find a 512x512 PNG image
  2. Go to https://convertio.co/png-ico/
  3. Upload your PNG and convert to ICO
  4. Download both files and place them here

Option 2 - Use GIMP (free software):
  1. Download GIMP from https://www.gimp.org/
  2. Create a 512x512 image
  3. Export as PNG (icon.png)
  4. Export as ICO (icon.ico)

Option 3 - Use Photoshop/Illustrator:
  1. Create your icon design
  2. Export as 512x512 PNG (icon.png)
  3. Use a plugin or online tool to convert to ICO

RECOMMENDED ICON DESIGN:
------------------------
- Simple, recognizable design
- Works well at small sizes (16x16, 32x32)
