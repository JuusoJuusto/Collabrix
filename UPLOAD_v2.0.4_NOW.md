## 🔍 TROUBLESHOOTING

### Update Not Showing?
1. Make sure you're on v2.0.3
2. Check internet connection
3. Restart Collabrix
4. Click "Check for Updates" in Settings
5. Wait 30 seconds for check to complete

### Download Fails?
1. Check firewall settings
2. Check antivirus (may block download)
3. Try manual download from GitHub
4. Run installer manually

### Update Installs But Version Still 2.0.3?
1. Completely close Collabrix
2. Check Task Manager (no Collabrix processes)
3. Relaunch Collabrix
4. Check version in Settings

### Features Not Working?
1. Hard refresh (Ctrl+Shift+R)
2. Clear cache
3. Restart app
4. Reinstall if needed
